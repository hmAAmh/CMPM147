Description

Reflection

I found this project interesting for how it gave me a deeper appreciation for the artworks I modified through it. In being able to break down the overall color choices, as well as the macro-scale shape of the piece as a whole, I was able to find new things to appreciate in the originals. Additionally, it emphasized to me the scale to which seemingly small changes in visual art can spiral into radically different outcomes.
