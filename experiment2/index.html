Living Impression

Description

This project offers a generative, living impression of this inspiring image:

Technical

Relation to inspiring image:

  • Color: The grass, stone, water, and lily pad colors were sampled from the inspiring image using the Photoshop eyedropper tool.
  • Primitives: The lily pads and rocks are rendered with `ellipse` primitives.
  • Polygons: The water is a single randomly polygon using `vertex` calls.
  • Composition: Different random seed values lead to offset movement on the lily pads and water, slightly responding to the time via Millis.
